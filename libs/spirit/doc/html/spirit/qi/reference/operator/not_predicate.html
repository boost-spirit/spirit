Not Predicate (operator!())
Description

Syntactic predicates assert a certain conditional syntax to be satisfied before evaluating another production. Similar to semantic predicates, eps, syntactic predicates do not consume any input. The not predicate, !, is a negative syntactic predicate that returns a zero length match only if its predicate fails to match.
