Auxiliary Generators

See here for more information about Auxiliary Generators.

Expression

Attribute

Description

attr_cast<Exposed>(a)

Exposed

Invokes a while supplying an attribute of type Exposed.

eol

Unused

Generates the end of line (\n)

eps

Unused

Generate an empty string

eps(b)

Unused

If b is true, generate an empty string

lazy(fg)

Attribute of G where G is the return type of fg

Invoke fg at generation time, returning a generator g which is then called to generate.

fg

see lazy(fg) above

Equivalent to lazy(fg)

g[fa]

Attribute of g

Call semantic action fa (before executing g).
