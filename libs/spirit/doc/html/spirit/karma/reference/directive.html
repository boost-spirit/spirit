This module includes different generator directives. It includes alignement directives (left_align[], center[], and right_align[]), repetition (repeat[]), directives controlling automatic delimiting (verbatim[] and delimit[]), controlling case sensitivity (upper[] and lower[]), field width (maxwidth[]), buffering (buffer[]), and attribute handling (omit[]).
