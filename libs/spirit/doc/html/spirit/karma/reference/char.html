Char

Character Generators
Character Class Generators

This module includes different character oriented generators allowing to output single characters. Currently, it includes literal chars (e.g. 'x', L'x'), char_ (single characters, ranges and character sets) and the encoding specific character classifiers (alnum, alpha, digit, xdigit, etc.).

Module Header 
// forwards to <boost/spirit/home/karma/char.hpp>
#include <boost/spirit/include/karma_char.hpp>
